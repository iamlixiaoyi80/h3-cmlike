# H3-CMlike 🏰

英雄无敌：王国征服 - CMlike（类 Coin Master）+ 英雄无敌3 题材

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 技术栈

- Vue 3 + TypeScript
- Pinia (状态管理)
- Vite (构建工具)

## 功能

- 🎲 骰子系统（可操作：点击掷出 → 旋转 → 点击停止）
- 🗺️ 环形地图（24格，7种事件类型）
- 🏰 3个种族城堡（骑士/精灵/亡灵）
- 🦸 9位英雄（每种族3位）
- 🐉 21种生物（每种族7级）
- ⚔️ 简单战斗系统
